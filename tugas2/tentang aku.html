Nama saya Anggy Oktaviana Syafira, biasa dipanggil Anggi. Saya lahir tanggal 25 Oktober 1999. Saya anak kedua dari dua bersaudara. Alamat saya di Purimas Jl. Taman Pecatu E9/24.

Saat ini saya berumur 20 tahun dan sedang menempuh pendidikan di Universitas Pembangunan Nasional "Veteran" Jawa Timur. Saya mengambil studi Sistem Informasi di Fakultas Ilmu Komputer.

Hobi saya mendengarkan lagu, membaca buku dan menulis. Genre musik favorit saya adalah pop, jaz, dan lo-fi. Musisi luar favorit saya adalah Rex Orange County, Jeff Bernat, The Walters dan Bruno Major. Saya juga suka mendengarkan lagu-lagu dari band lawas seperti The Beatles dan Westlife. Untuk musisi lokal sendiri, saya suka mendengarkan lagu-lagu dari Ardhito Pramono, Tulus, Saint Fiction, dan Hivi. Selain mendengar musik, saya juga suka mendengarkan podcast. Untuk mengetahui lebih banyak tentang selera musik saya, teman-teman bisa melihat playlist musik saya pada menu di beranda.

Mengenai buku, saya suka membaca buku-buku fiksi maupun non fiksi. Sejak kecil ssaya suka mengoleksi buku-buku komik seperti serial Why, Seri Tokoh Dunia, juga buku novel anak-anak. Untuk sekarang, saya suka membaca buku-buku novel, terutama karangan Tere Liye. Saya juga suka menulis, dan baru-baru ini saya juga mulai menulis di blog, silahkan kunjungi Tulisannya Anggik
